# 🏦 Sistema Bancário em Java

🚀 Projeto de estudo em **Java**, focado em **POO** (herança, polimorfismo, encapsulamento, classes abstratas) com menu interativo para operações bancárias.

---

## 🔹 Descrição

Este projeto simula um **sistema bancário** simples, permitindo:

- Criar contas correntes
- Realizar depósitos e saques
- Consultar extratos

O sistema foi desenvolvido com foco em **Java básico/intermediário**, aplicando conceitos de **orientação a objetos**:

- Classes abstratas e herança (`Conta` abstrata e `ContaCorrente`)  
- Polimorfismo e sobrescrita de métodos (`sacar`)  
- Encapsulamento com getters e setters  
- Estrutura de dados `ArrayList` para extrato  
- Menu interativo usando `Scanner`

---

## ⚙️ Funcionalidades

- Criar conta corrente com limite de cheque especial e taxa de saque  
- Depositar em contas existentes  
- Sacar dinheiro considerando saldo + limite especial  
- Consultar extrato de operações realizadas  

---

## 🛠️ Tecnologias

- Java 17+  
- Programação Orientada a Objetos (POO)  
- Estruturas de dados: `ArrayList`  
- Entrada de dados: `Scanner`
